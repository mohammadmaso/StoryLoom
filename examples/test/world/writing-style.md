# Writing Style

## Style Profile

تحلیل یک فصل از متن اصلی نشان می‌دهد که میانگین طول جملات ۰.۱ کلمه است. نسبت دیالوگ ۰.۰٪ از خطوط را تشکیل می‌دهد. گرایش دیدگاه (POV) نامشخص است. واژگان رایج شامل «chapter»، «jhoi»، «plot»، «notes» و «charactersjhoi» هستند. این آمار نشان‌دهنده ساختار غیرمعمول و نیاز به بازبینی در انسجام متن است.
